## ğŸ’¡ Key Design Decisions

1. **Single-page app without a framework** â€” All views are rendered by swapping `innerHTML`, avoiding the complexity of React/Vue while keeping the game fast.

2. **Synthesized audio** â€” Instead of loading MP3 files, all sounds are generated live using the Web Audio API, making the game lightweight with zero audio assets.

3. **Offline-first architecture** â€” The game works fully offline using `localStorage`. The Flask backend is optional and adds multi-device sync and leaderboards.

4. **Progressive difficulty** â€” Each game gets harder as the player improves, keeping engagement high without overwhelming beginners.

5. **Combo & reward systems** â€” Coins, stars, combos, and speed bonuses create a satisfying feedback loop that motivates continued play.
